EXAMPLES OF MY WORK

Below you can see a few samples of the work I have done together with a short description of the project, it's requirements and a view of the project. To see the live website, just press the image! If you are interested in examinging the code for the project, feel free to check out my GitHub.

Spotify

This project consisted of creating what would look like a Spotify music player. I was told to open the Spotify music player on my laptop and to recreate the layout from scratch without any design guidelines. This was my first attempt at creating something from scratch without having the guidelines from Figma to help along the way. This project was a great learning experience because it forced me to really think about how the music player was laid out. I was not given the layout answers straight away as with most other designs through Figma, but required me to be creative in how to approach the construction of the site. Not that the site does not actually play music since this was an exercise in design layout, not audio. However, if I were to remake this project, I would definitely work on adding audio to it. Note that this is only developed for desktop.

Bhromaon Travel

This project focused on making a functioning website that incorporated most of the basics of website development. The website was created from scratch based on a design in Figma. It was to be fully responsive, have several images and sections with content, and be completed within a narrow timelimit. This was one of the more complex designs I have (at the time) created and am happy with how the result turned out. This project pushed me to think quickly and creatively on how to solve the problem at hand: how do I create this large landing page as complete as possible within my timelimit? This was a good challange in finding a balance between getting the page to look good and having the page be complete with all sections, information, and functionality.

Patagonia

This project focused on making a site responsive to different screen sizes. I was given the design in Figma with three different sizes (desktop, tablet, and mobile). I then recreated the design from scratch and implemented the necessary flexible code and media queries to make the website look good not only on the three different device screen sizes, but on all sizes inbetween as well.

Genesis Portfolio

This project focused on creating a layout for a portfolio of photographs and using Google Maps. The project was created from scratch based on a design in Figma. The project also links between three separate pages on the website. This project was good practice for making separate pages of a site look cohesive and how to use flexbox and grids in the most optimal ways in order for the site to look good on all screen sizes.
